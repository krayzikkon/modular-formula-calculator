# math-expressions-calculator
Простой проект для вычисления двух математических выражений (A и B) по заданным формулам. Разработан как учебный пример модульной структуры проекта.

# Инструкция по сборке и запуску

1. Откройте терминал в Linux
2. Перейдите в папку с проектом
3. Выполните команду `make` для сборки проекта
4. Запустите программу: `./calculator`
5. Для очистки проекта: `make clean`

# Инструменты среды Linux

1. **vim** - текстовый редактор для написания кода
2. **g++** - компилятор C++
3. **make** - утилита для автоматизации сборки
4. **gdb** - отладчик для поиска и исправления ошибок
5. **bash** - командная оболочка для работы с терминалом

# Отладка с помощью gdb

Если программа работает некорректно:
1. Скомпилируйте с флагом -g: `make`
2. Запустите отладчик: `gdb ./calculator`
3. Используйте команды gdb:
   - `break main` - установить точку останова
   - `run` - запустить программу
   - `print variable_name` - просмотреть значение переменной
   - `step` - выполнить шаг программы
   - `continue` - продолжить выполнение
   - `quit` - выйти из gdb

# Пример выполнения

После запуска программа должна вывести:
```
For x = -0.620000, y = 0.820000, z = 25.000000:
A = 0.024220
B = 0.678126
```

Этот проект демонстрирует модульную структуру, где функции A и B вынесены в отдельные файлы, и использует стандартные инструменты разработки в Linux.
